# Matip Estimate Converter (PWA)

## 概要
見積書を構造化データに変換するAIツールのPWA版です。

## GitHub Pages での公開手順
1. このフォルダの内容を GitHub リポジトリにアップロードします。
2. リポジトリの **Settings** > **Pages** に移動します。
3. **Build and deployment** の **Source** で `Deploy from a branch` を選択します。
4. **Branch** で `main` (または `master`) と `/ (root)` を選択して **Save** をクリックします。
5. 数分後、表示された URL でアプリが利用可能になります。

## PWA機能
- オフライン対応（キャッシュ）
- ホーム画面への追加
- スタンドアロン起動
